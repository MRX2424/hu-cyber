Three Tracks, Nine Levels

Each track builds depth with three levels. Complete lessons, labs, and quizzes in Level 1 before advancing to Level 2 and 3. Use the outcome statements and FAQs below to choose your next steps.

Networking for Web

20 labs

Understand the moving parts of the web: HTTP verbs, headers, TLS, redirects, and reconnaissance workflows.

Level 1
HTTP basics, DNS, DevTools, Burp proxy, sessions, TLS primers.
Level 2
Nmap fingerprinting, Wireshark capture, CORS, rate limiting, log review.
Level 3
Recon-to-exploit scenarios, request smuggling intro, disclosure practice.
Track details →

Kali Linux & Tools

20 labs

Become fluent in Kali, Burp Suite, FFUF, and other platform essentials that power modern pentests.

Level 1
Kali setup, terminal literacy, core enumeration utilities, browser tooling.
Level 2
Intruder payloads, SQLmap assist, tool chaining, CSRF workflows.
Level 3
Automation macros, WAF evasion, multi-step exploit labs, log parsing.
Track details →

OWASP / Web Exploits

20 labs

Investigate common vulnerabilities, exploit chains, and responsible remediation guidance.

Level 1
OWASP Top 10 overview, XSS varieties, intro SQLi, LFI, session hygiene.
Level 2
Blind SQLi, IDOR, SSRF, DOM XSS, business logic flaws, safe chaining.
Level 3
End-to-end pentest testbed, deserialization, WAF evasion, reporting.
