Track

Networking for Web

Translate packets to pentests. Build a strong foundation in how browsers, proxies, and servers communicate so you can recognize anomalies and exploitation opportunities.

Levels & outcomes

Level 1

Foundation

Master HTTP requests, status codes, headers, cookies, and TLS basics.

  • Lesson: HTTP Basics
  • Lesson: Burp Proxy Intro
  • Lab: Capture & Replay
  • Quiz: Networking L1 quiz

Level 2

Intermediate

Pivot from packet capture to analysis. Study CORS, redirects, and rate limiting strategies.

  • Lesson: Nmap fingerprinting workflow
  • Lab: Wireshark request trace
  • Lesson: CORS misconfig overview
  • Quiz: End-of-level scenario quiz

Level 3

Advanced

Combine reconnaissance, exploitation, and responsible disclosure in capstone-ready engagements.

  • Scenario lab: Recon → exploit chain
  • Case study: HTTP request smuggling
  • Template: Disclosure email walkthrough
