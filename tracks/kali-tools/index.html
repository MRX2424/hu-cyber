Track

Kali Linux & Tools

Learn the everyday toolkit of a web pentester. Configure Kali, streamline Burp Suite workflows, and chain CLI tools without needing to code.

Levels & outcomes

Level 1

Setup

Install Kali, use the terminal confidently, and capture traffic with Burp & browser extensions.

  • Lesson: Kali onboarding checklist
  • Lab: Enumerate with Nmap + FFUF
  • Lesson: Burp Repeater reps
  • Quiz: Tool identification & use

Level 2

Workflow

Automate repetitive actions with Intruder, SQLmap assist, and tool chaining for CSRF verification.

  • Lesson: Intruder payload design
  • Lab: SQLmap safe practice
  • Lesson: Token workflows
  • Quiz: Toolchain sequencing

Level 3

Advanced

Leverage macros, evasions, and log analysis to sustain longer engagements and produce exportable evidence.

  • Lesson: Burp macros & sessions
  • Lab: WAF evasion drills
  • Lesson: Forensic log parsing
