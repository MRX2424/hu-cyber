Burp Suite Proxy Fundamentals

Configure interception, scope traffic, and replay requests with Burp Suite Community Edition.

Objectives

  • Install Burp CA certificate in Firefox.
  • Configure proxy listener and browser settings.
  • Intercept and forward traffic safely.
  • Send requests to Repeater for modification.

Overview

Burp Suite sits between your browser and the target application, allowing you to observe and modify requests on the fly. Proper setup ensures encrypted traffic is decrypted ethically within your local environment. You will configure certificates, define target scope, and practice forwarding vs. dropping requests to maintain control while testing.

Demo steps

  1. Launch Burp Suite → Proxy → Options → Confirm listener on 127.0.0.1:8080.
  2. In Firefox, set manual proxy to 127.0.0.1 port 8080 and import Burp certificate from http://burp.
  3. Visit an allowed lab URL, observe interception, toggle intercept off, and send copies to Repeater.
  4. Use Target → Scope to restrict to lab hostname and avoid capturing unrelated traffic.
